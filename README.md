# MCEconomy Plugin Website

Static documentation site for the MCEconomy plugin ecosystem.  
Hosted via GitHub Pages from the `docs/` directory.

## Structure

```text
docs/
├── api/            # API module documentation
├── common/         # Common library documentation
├── platforms/      # Platform-specific documentation (PaperMC, etc.)
├── css/            # Shared and partial stylesheets
├── js/             # Client-side scripts
├── partial/        # Reusable HTML partials (header, footer)
└── index.html      # Documentation entry page
```

## Deployment

This project uses the `docs/` directory as the publish source for GitHub Pages.  
Any push to the default branch will automatically update the live site.
